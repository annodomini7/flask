Инструкция по отправке остатков препаратов


Для того, чтобы наши чат-боты выдавали пользователям актуальную информацию, Вам необходимо присылать информацию об остатках препаратов в апетке в формате .json. На страничке Вашего профиля есть ссылка, по которой необходимо присылать файл. Структура файла должны быть следующей:

{
	"password":"your password",
	"data":[
		{
			"barcode":4607013581074,
			"cost":130,
			"number":1

		},
		{
			"barcode":4605964004123,
			"cost":267,
			"number":1

		},
		{

			"cost":399,
			"number":1

		}
	]
}
